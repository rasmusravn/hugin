# Visual Concept Prompts for Hugin
These prompts are designed for generative AI image tools to create conceptual artwork for the Hugin project.

---

## Prompt 1: The Hugin Ball (Detailed Technical)

```
A transparent spherical sensor ball cutaway view showing internal electronics. Inside the clear sphere:
- A Raspberry Pi Pico W microcontroller board (green PCB with wireless antenna symbol)
- An MPU6050 IMU sensor module (small blue breakout board with 6-axis motion tracking)
- A VL53L1X Time-of-Flight distance sensor (small purple board with clear sensor window pointing outward)
- A small LiPo battery pack (silver rectangular pouch)
- Thin colored wires connecting components (red for power, black for ground, blue and yellow for I2C data lines)

The sphere is approximately tennis ball sized (6-7cm diameter). The ToF sensor has a clear window in the sphere's surface pointing forward. Small mounting posts visible inside holding components in place. Professional technical illustration style with labels pointing to each component. Clean white background. Isometric 3/4 view showing the interior layout clearly.
```

## Prompt 4: Component Breakdown (Educational)

```
An exploded view technical diagram of the Hugin sensor ball showing all components separated along a vertical axis:

From top to bottom:
1. Transparent hemisphere (upper shell) - clear plastic with mounting bosses
2. VL53L1X ToF sensor - small purple breakout board with clear sensor lens
3. MPU6050 IMU - small blue/green breakout board
4. Raspberry Pi Pico W - green circuit board with RP2040 chip visible and WiFi antenna
5. LiPo battery - silver pouch cell with JST connector
6. Transparent hemisphere (lower shell) - matching bottom half

Thin dotted lines connecting each component to its position in the assembled sphere (shown as ghost outline on right side). Each component has a label with name and simple icon. Clean white background. Orthographic projection. Technical manual style with precise spacing between components. Scale bar showing 5cm reference.

Color scheme: Components in realistic colors, shells in light blue transparency, labels in dark gray text, connection lines in thin orange dashes.
```

## Usage Notes

- These prompts work best with AI image generators like Midjourney, DALL-E 3, or Stable Diffusion
- Adjust technical detail level based on intended audience (technical vs. general public)
- Prompts 1-4 are more educational/documentation focused
- Prompts 5-6 are more marketing/artistic focused
- Consider generating multiple variations and combining elements from different results
